# 📈 AI日経225 株価予測＆バックテストシステム

日経225採用銘柄を対象に、機械学習（ランダムフォレスト）を用いて翌日の株価動向（上がるか・下がるか）を予測し、有望銘柄を毎日Discordに自動通知するシステムです。
単なる予測モデルの構築に留まらず、過去データに基づく独自のバックテストエンジンを実装し、実運用における期待値やリスクを定量的に検証できる実践的な設計としています。

## 📒 開発の背景と目的
- **開発のきっかけ**: これまで手作業で行っていたトレード戦略の収益シミュレーションを自動化し、より客観的かつ膨大なデータに基づいた検証を行いたいと考えたためです。
- **開発のステップ**:
  1. Jupyter Notebookを用いたデータ収集・前処理・特徴量エンジニアリング・モデル構築のプロトタイプ作成
  2. ルールベースの売買シミュレーション（バックテスト）機能の実装
  3. 保守性と拡張性を考慮したオブジェクト指向（クラス化）によるコードのリファクタリング
  4. 予測精度向上のための分析対象銘柄の拡大および、特徴量の追加

## ✨ 主な機能と工夫点
- **データパイプラインの自動化**: `yfinance` API を活用し、日経225銘柄および主要マクロ指標（S&P500, VIX, 為替など）のデータを自動取得・統合。
- **実践的な特徴量エンジニアリング**: 
  - **データリーク対策**: 未来情報の混入を完全に防ぐための `shift()` 処理を徹底。
  - **相対モメンタムの考慮**: 東証33業種と連携した「セクター平均騰落率」や「セクターからの出遅れ乖離率」など、相場環境を加味した高度な特徴量を生成。
- **機械学習モデル (予測)**: `scikit-learn` の Random Forest Classifier を使用し、翌日の株価（始値➔終値）がプラスになる確率をスコアリング。
- **自作バックテストエンジン**: 
  - 「寄り買い・引け売り」「寄り空売り・引け買い戻し」戦略のシミュレーション。
  - 実際のトレードを想定し、利確・損切りラインを設定した OCO（One Cancels the Other）決済ロジックの実装。
- **Discord自動通知機能**: 毎日のバッチ処理により最新の予測スコアを計算し、買い候補・空売り候補のトップ10をWebhook経由で自動配信。

## 🛠️ 技術スタック
- **言語**: Python 3.11
- **データ処理**: Pandas, NumPy
- **機械学習**: scikit-learn (RandomForest)
- **外部通信・データ取得**: yfinance, pandas_market_calendars, requests
- **アーキテクチャ**: オブジェクト指向設計（クラス化による保守性・再利用性の向上）
- **その他**: `logging` モジュールによる運用ログの記録
