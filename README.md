# 📈 AI日経225 株価予測＆バックテストシステム

日経225採用銘柄を対象に、機械学習（ランダムフォレスト）を用いて翌日の株価動向（上がるか・下がるか）を予測し、有望銘柄を毎日Discordに自動通知するシステムです。
単なる予測だけでなく、過去データに基づくバックテストエンジンを自作し、実運用における期待値を検証できる設計になっています。

## 📒 作成の経緯
- **きっかけ**: 手動でしていたシミュレーションによる収益率を自動で行うためにPythonでの開発に取り掛かりました。
- **経緯**:
  - JupyterNotebookでのデータ収集・前処理・特徴量エンジニアリング・予測を作成
  - ルールどおりの売買での収支を計算するためバックテスト計算を実装
  - クラス化でリファクタリング実施
  - 特徴量と学習データを増やすため分析銘柄変更
  
## ✨ 主な機能
- **自動データ収集**: `yfinance` API を活用し、日経225銘柄および主要指数（S&P500, VIX, 為替など）のデータを自動取得。
- **時系列特徴量エンジニアリング**: 
  - データリーク（未来情報の混入）を完全に防ぐ `shift()` 処理の実装。
  - 東証33業種と連携した「セクター平均騰落率」や「セクターからの出遅れ乖離率」など、高度な相対モメンタム特徴量を生成。
- **機械学習モデル (予測)**: `scikit-learn` の Random Forest Classifier を使用し、翌日の株価（始値➔終値）がプラスになる確率をスコアリング。
- **バックテストエンジン**: 
  - 「寄り買い・引け売り」「寄り空売り・引け買い戻し」戦略のシミュレーション。
  - 利確・損切りラインを設定した OCO（One Cancels the Other）決済の実装。
- **Discord自動通知**: 毎日計算された最新の予測スコアから、買い候補トップ10・空売り候補トップ10を抽出し、Webhook経由で自動配信。

## 🛠️ 技術スタック
- **言語**: Python 3.11
- **データ処理**: Pandas, NumPy
- **機械学習**: scikit-learn (RandomForest)
- **データ取得**: yfinance, pandas_market_calendars, requests
- **アーキテクチャ**: オブジェクト指向設計（クラス化による保守性・再利用性の向上）, `logging`による運用ログの記録
